# 🚀 DASAR INSTALL VPS

Script dasar untuk setup VPS + Tailscale

---

## 🔹 1. Setup VPS

```bash
chmod +x setup1-vps.sh
./setup1-vps.sh
```

---

## 🔹 2. Install Tailscale

```bash
chmod +x tailscale/tailscale-openclaw.sh
./tailscale/tailscale-openclaw.sh
```

---

## 🔹 Atau langsung download tanpa git clone

### Setup VPS:
```bash
curl -O https://raw.githubusercontent.com/ardiyan2/DASAR-INSTALL-VPS/main/setup1-vps.sh
```

### Tailscale:
```bash
curl -O https://raw.githubusercontent.com/ardiyan2/DASAR-INSTALL-VPS/main/tailscale/tailscale-openclaw.sh
```
